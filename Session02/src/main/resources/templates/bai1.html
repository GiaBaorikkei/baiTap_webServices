JPA (Java Persistence API) là một đặc tả (specification) trong Java dùng để quản lý dữ liệu giữa ứng dụng Java và cơ sở dữ liệu quan hệ. JPA không phải là một framework, mà là một tiêu chuẩn do Java EE định nghĩa, cho phép lập trình viên thao tác với dữ liệu bằng cách ánh xạ (mapping) các đối tượng Java sang bảng trong cơ sở dữ liệu. Hibernate là một trong những implementation phổ biến nhất của JPA, cung cấp đầy đủ các tính năng của JPA và mở rộng thêm nhiều tiện ích hữu ích khác. JPA giúp giảm sự phụ thuộc vào các câu lệnh SQL thuần túy, đơn giản hóa việc truy cập và xử lý dữ liệu, đồng thời hỗ trợ tự động tạo bảng, truy vấn linh hoạt và quản lý vòng đời entity. Nhờ đó, JPA đóng vai trò quan trọng trong việc phát triển các ứng dụng Java hiện đại, giúp mã nguồn dễ bảo trì, linh hoạt và dễ mở rộng hơn.
