Tiêu chí Spring MVC Spring Boot
Khái niệm Là một module trong Spring Framework dùng để xây dựng ứng dụng web theo mô hình MVC truyền thống. Là một framework được xây dựng trên nền tảng Spring, giúp phát triển ứng dụng Spring dễ dàng và nhanh chóng.
Cấu hình Cần cấu hình nhiều file XML hoặc Java để thiết lập các thành phần của ứng dụng. Có cấu hình tự động, giảm thiểu việc phải viết mã cấu hình, sử dụng các annotation để cấu hình.
Khởi tạo dự án Phải tạo cấu trúc và file cấu hình thủ công hoặc sử dụng Spring Initializr kèm các cài đặt riêng. Có thể khởi tạo nhanh chóng bằng Spring Initializr, hỗ trợ cấu hình sẵn.
Quản lý phụ thuộc Người dùng phải chọn và cấu hình các dependency phù hợp. Spring Boot cung cấp starter dependencies giúp quản lý phụ thuộc dễ dàng hơn.
Thời gian phát triển Tốn thời gian do phải thiết lập nhiều cấu hình ban đầu. Tiết kiệm thời gian phát triển nhờ cấu hình tự động và starter packages.
