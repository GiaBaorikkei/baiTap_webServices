RESTful API (Representational State Transfer) là một kiến trúc thiết kế dùng để xây dựng các dịch vụ web, hoạt động dựa trên giao thức HTTP. Một trong những nguyên lý quan trọng của REST là stateless – nghĩa là mỗi request từ client phải chứa đầy đủ thông tin để server hiểu và xử lý, server không lưu trạng thái của client giữa các request. REST cũng tuân thủ nguyên tắc client-server, tức là client và server tách biệt nhau: client chỉ quan tâm đến giao diện và thao tác, còn server xử lý dữ liệu và logic. RESTful API sử dụng các HTTP methods như GET (lấy dữ liệu), POST (tạo mới), PUT (cập nhật), và DELETE (xóa dữ liệu), giúp phân biệt rõ ràng các hành động thao tác trên tài nguyên (resource). Nhờ tính đơn giản, dễ mở rộng và tương thích cao, REST được sử dụng rộng rãi trong các ứng dụng web và mobile hiện nay.
